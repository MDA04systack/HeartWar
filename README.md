# HeartWar (하트워)

**아시아경제 교육센터 헬스케어 4기 파이게임 프로젝트 (1팀)**
- **팀장:** 박수연 (DA)
- **팀원:** 김용기(AA), 유성빈(TA), 박윤정(DA)

## 1. 프로젝트의 주요 기능과 목적

### 개요
**HeartWar**는 심근경색 인식 개선을 위한 간단한 벽돌 깨기 게임입니다. 
지질 축적은 심근경색 및 심혈관 질환의 주요 원인 중 하나로 알려져 있습니다. 본 게임에서는 지질을 게임 속 방해꾼(벽돌)으로 설정하여, **"지질 = 위험 요소"**라는 개념을 직관적으로 이해할 수 있도록 돕습니다.

### 목적 및 교육적 의미
- 게임을 플레이하는 사람들에게 심근경색에 좋은 영양소를 기억에 남게 하기 위함
- 심근경색이 오기 전에 지질의 방해를 피해 혈관 내 벽돌들을 제거하는 스토리텔링 적용

### 주요 기능
- **아이템 시스템:** 심근경색에 좋은 영양소를 게임 클리어에 도움이 되는 아이템으로 설정하여 획득 시 이로운 효과 제공
- **필살기:** 방해 요소(지질 벽돌)를 한 번에 대량 처리할 수 있는 스킬 기능
- **제한 시간 시스템:** 플레이어가 250초의 제한 시간 내에 스테이지를 클리어하도록 설정하여 긴장감 부여
- **방해 시스템:** 혈관 내 지질을 방해 요소로 출현시켜 타격 및 회피의 재미요소 추가

**[게임 플레이 플로우차트]**
- 제한 시간 타이머 및 위험 요소 출현을 위한 타이머 작동
- 게임 진행: 공 상하좌우 이동, 패들(Paddle) 좌우 움직임 조작
- 충돌 감지: 벽돌인 경우와 아닌 경우로 분류하여 공이 튕기는 각도 및 처리 적용
- 생명 차감: 공이 패들에 닿지 못하고 바닥에 떨어졌을 경우 생명 1개 차감

## 2. 설치 방법

본 프로젝트는 **Python 3.12** 환경과 **Pygame** 라이브러리를 사용합니다.

1. **저장소 클론(Clone)**
   ```bash
   git clone https://github.com/MDA04systack/HeartWar.git
   cd HeartWar
   ```

2. **가상환경 생성 및 실행 (선택 사항)**
   ```bash
   python -m venv venv
   # Windows의 경우
   venv\Scripts\activate
   # macOS/Linux의 경우
   source venv/bin/activate
   ```

3. **필요한 라이브러리 설치**
   ```bash
   pip install -r requirements.txt
   ```
   *(이 과정에서 핵심 의존성인 `pygame`이 설치됩니다.)*

4. **게임 실행**
   ```bash
   python arkanoid.py
   ```

## 3. 문제 해결 방법 (Troubleshooting)

- **`pygame` 모듈을 찾을 수 없다는 에러가 발생할 때:**
  `pip install pygame` 명령어를 통해 파이게임을 수동으로 설치하거나, `pip install -r requirements.txt` 명령어를 다시 확인 후 실행해 주세요.

- **화면이 깨지거나 실행 창이 바로 닫힐 때:**
  권장하는 파이썬 버전인 `Python 3.12` 환경에서 실행해 주세요. 또한 소스 코드나 이미지의 리소스 경로가 올바르게 위치해 있는지 확인해 주세요.

- **조작키가 반응하지 않을 때:**
  키보드의 한/영 상태가 '한글'인지 확인해 주세요. 원활한 조작을 위해 '영문' 상태로 두고 플레이하시길 권장합니다.
